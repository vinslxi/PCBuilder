Before You Start Building

  • Set up a clean, static-safe workspace.
  • Gather tools: Phillips screwdriver, flashlight, and cable ties.
  • Read the motherboard manual for CPU socket, RAM slots, and front-panel connections.

High-Level Build Steps

  1. Install the CPU and cooler on the motherboard.
  2. Install RAM and any M.2 NVMe SSDs.
  3. Mount the motherboard in the case and connect the 24-pin and CPU power cables.
  4. Install the power supply and route cables cleanly.
  5. Install the GPU and connect PCIe power cables if required.
  6. Connect case fans, storage drives, and front-panel connectors.
  7. Power on the system, enter BIOS, and enable XMP/EXPO for RAM if supported.
  8. Install Windows and required drivers.

First Boot Checklist

  • All fans are spinning and no loose cables touch the blades.
  • The display is connected to the GPU (if present) rather than the motherboard.
  • Temperatures look normal in hardware monitoring software.
  • The system can restart several times without errors.

Before You Begin

  • Clear a static-safe workspace; gather a Phillips screwdriver and motherboard manual.
  • Verify CPU socket, RAM type, case size, and PSU connectors.

Build Steps (High-Level)

  1. Install CPU and cooler on the motherboard.
  2. Insert RAM and M.2 NVMe SSD.
  3. Mount motherboard in the case; route 24-pin and EPS cables.
  4. Install PSU and GPU; connect PCIe power and front-panel headers.
  5. First boot to BIOS; enable XMP/EXPO; install Windows and drivers.

Troubleshooting

No display on first boot?
Reseat GPU and RAM; verify monitor is on GPU HDMI/DP; check 8-pin CPU power.
High CPU temps?
Re-apply thermal paste; tighten cooler evenly; confirm fan direction and curve.
PC shuts off under load?
Check PSU wattage and cables; update GPU drivers; test RAM with MemTest.
